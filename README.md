# Study Planner

A web-based Study Planner system developed using PHP and MySQL.

## Features

- User Registration & Login
- Forgot Password via Email OTP
- Dashboard Statistics
- Subject Management
- Task Planner
- Reminder System
- Progress Tracking
- Rewards System
- Profile Management
- PDF Report Export
- Admin Panel
- Contact Us Page
- About Us Page
- Email Notifications

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- PHPMailer
- FPDF

## Installation

1. Copy project to htdocs.
2. Import database SQL file.
3. Configure config.php.
4. Start Apache and MySQL.
5. Open localhost/study_planner.

## Admin Access

Set a user as admin:

```sql
UPDATE users SET is_admin = 1 WHERE id = 1;
```

## SMTP Configuration

Configure SMTP settings from the Admin Settings panel.
